# CoreGearTech

E-commerce website with search and cart using PHP and MySQL.

E-commerce website project for Core Gear Tech.

## Setup
1. Configure your web server to serve this project.
2. Create and import the database schema from `schema.sql`.
3. Update any environment or database settings as needed.

## Notes
- This repository contains PHP files for the frontend and backend.
- Static assets and libraries are included in the project tree.
